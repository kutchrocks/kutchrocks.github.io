3.1 Vegas Background

Vegas is plugin to set up a background with different types of overlays in our theme. You can find complete documentation here.

To change background image you simply open folder “image” and paste your image here. Your image has to be in .jpg format and should be named „background“.

If you have another image format go to folder “js” and open file “custom.js”. In this file you have to change line number 13

src:'assets/images/background.jpg'

Here you can edit image format. For example like this:

src:'assets/images/background.png'

Choose your favourite overlay

Vegas plugin allows you to change different types of overlay. To do that you have to go to folder “js” open “custom.js” file and edit line number 14:

src:'assets/images/overlays/05.png'

Vegas plugin has 15 different types so it’s up to you which you choose e. g.

src:'assets/images/overlays/15.png'

3.2 Font Awesome

It’s one of the most powerfull tools. Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS. Let’s take a look. In the index.html you can find few icons in Footer. This is a font awesome so let’s say we wanna change them to more suitable for your purposes. Open the index.html in some editor and find following code.

<i class="fa fa-facebook"></i>

Here you can change appearance of this icon just simply remove class „fa-facebook“ and type for instance „fa-twitter“. Now save the changes and refresh page and icon is different.

For full documentation how to customize this plugin (size, color, rotation etc.) and full list of different types of icons go to fontawesome.io

3.3 Countdown circles

Amazing plugin that allows us countdown with circle style. You can find complete documentation at git.wimbarelds.nl

To set up time to countdown you just have to open index.html in some editor. Then on the line 161 set up your date. The format is yyyy-mm-dd.

4.1 Google Map

To change location in Google maps leads several steps:

  1. Open browser -> go to maps.google.com
  2. Find location which you want show in your webpage
  3. In the URL row of your browser copy the numbers after @ in this case "43.651071,-79.378764"
  4. Open custom.js file go to line 52(53) and paste GPS coordinations
  5. You can also change zoom of map (line 54)

You can also change map marker title (line 65)
